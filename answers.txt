Q1: Why might we consider using Git?

Answer: Git is a powerful version control tool that is widely used in industry, in both software engineering and data science. It's important for data scientists to know how to collaborate with each other, and Git is one popular way of achieving that seamless collaboration.

Q2: What field could data science be applied to? Give an Example.

Answer: E-commerce. Data science could be used to personalize recommendations.

Q3: What is the difference between Merging and Rebasing?

Answer: Merging links the history of branches, while rebasing creates new commits for the commits of the original branch. Merging results in a branching history that eventually joins together again, while rebasing results in one linear branch.
